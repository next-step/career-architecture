# career-architecture
- 이름: 박상권

## 🚀미션
- 결제 취소 기능 마이그레이션

### 개선포인트 분석
- 취소 기능에 대한 레거시 코드에서 계속해서 취소 기능에 대한 개발이 이루어져 현재 유지보수가 불가한 상태
- 이 상태에서도 추가적인 개발이 일어나면 취소 기능은 어쩔 수 없이 기능 개발이 일어나고 있는 상태
- 그로인해서 취소 기능과 관련된 여러 장애가 발생해도 걷잡을 수 없는 상태
